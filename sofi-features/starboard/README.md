# Starboard

<figure><img src="../../.gitbook/assets/image (115).png" alt=""><figcaption></figcaption></figure>

## **How to Set Up?**

* Type `/serverconfig`&#x20;
* Go to **Starboard Setting (Sofi)**  menu
* and select #channel which you want to make a starboard channel.&#x20;
* Set active when you want to auto-add.
* Set the minimum number of wishlists that you want to add. (Premium Only)
